📚 Sobre o Projeto
---
Este aplicativo foi desenvolvido para ensinar, de forma interativa e divertida, os principais conceitos sobre artigo de opinião.
O aprendizado acontece através de mini quizzes ao longo do conteúdo e termina com um quiz final, que avalia o conhecimento adquirido pelo usuário.

O objetivo é transformar o estudo da produção textual em uma experiência mais dinâmica, incentivando o raciocínio crítico e a interpretação de argumentos.
